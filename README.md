### About

I run an agency that ships full-stack systems end-to-end: data pipelines, AI-powered conversational
funnels, payment integrations, ad-tracking infrastructure, and the dashboards on top of all of it.
I do the architecture, write the code, run the deploys, and own production.

Most of what I build hits real customers in days, not months — small surface, fast iteration,
shipped behind a load balancer with proper observability.

```
focus       full-stack engineer + ops, founder mindset
domains     conversational AI, lead generation, CRMs, payments, ad-tracking
philosophy  ship fast, instrument everything, refactor when it hurts — not before
```

### Things I care about

- **Owning the whole pipeline** — from ad click → landing → CRM → AI conversation → payment → fulfillment → tracking. Every gap is a leak.
- **Production observability** — health endpoints, structured logging, alerts, automatic backups, postmortems with prevention rules.
- **Security as default** — secrets in env vars, dedicated DB users with minimal grants, JWTs with rotation, branch protection, history hygiene (`git filter-repo` when it matters).
- **Pragmatic refactoring** — three similar lines beat a premature abstraction. Refactor when the cost of *not* refactoring is higher.
- **Working with AI as a tool** — fast iteration with LLMs, but discipline on plans, tests, and verification before completion. Vibe-coding doesn't ship.
