# Sergio Lima

**Full-stack developer** building production SaaS from zero.

Currently focused on **WhatsApp automation**, **AI-powered funnels**, and **fintech CRMs** for the Brazilian market.

## What I build

**Checkup360** — WhatsApp-first credit consultation platform
- Automated lead qualification with Gemini AI + Flowise
- WhatsApp audio transcription and natural conversation
- RPA for credit report generation (Puppeteer/KSI)
- Mercado Pago payments, Meta Pixel + CAPI tracking
- `Node.js` `Express` `TypeScript` `PostgreSQL` `Docker`

**Revenda Leads** — Lead resale CRM with operator management
- Kanban pipeline with drag-and-drop stage transitions
- Round-robin distribution with atomic balance debit
- Operator scoring, financial ledger, admin dashboard
- Meta Ads integration with server-side CAPI
- `Next.js` `React` `PHP` `Drizzle ORM` `PostgreSQL` `Docker`

## Stack

```
Backend:    Node.js, Express, Next.js, TypeScript, PHP
Database:   PostgreSQL, Redis, Drizzle ORM
Frontend:   React, Tailwind CSS, shadcn/ui, Vanilla JS
AI/ML:      Gemini 2.5 Flash, Flowise, RAG
Automation: Puppeteer, PM2, N8N
Infra:      Docker Swarm, Traefik, Nginx, Cloudflare
Payments:   Mercado Pago
Messaging:  Meta WhatsApp Cloud API, Evolution API
Tracking:   Meta Pixel, GA4, CAPI server-side
```
